### How to clone the project 

git clone https://github.com/shriyaagrawal08/LangChain.git
cd LangChain
code .

### Run these below in terminal

1) pip install uv

2) uv init

3) uv sync  // will create a virtual env

4) git add .

5) git commit -m "initial commit"

6) uv add langchain

7) source .venv/bin/activate  // only for MAC

8) touch .env
